# GoBarber

Projeto do Bootcamp GoStack da Rocketseat.

## Instalação

```
$ git clone git@github.com:cleiver/GoBarber.git

$ cd GoBarber

$ yarn

$ docker run --name postgres_gobarber -e POSTGRES_PASSWORD=docker -p 5432:5432 -d postgres

$ yarn dev:server
```
